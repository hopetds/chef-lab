## Steps: 
 - cd /opt/chef
 - wget https://github.com/chef-cookbooks/firewall/archive/master.zip
 - unzip master.zip
 - cd master
 - berks init
 - berks install
 - berks package
 - tar -xf cookbooks-1491400802.tar.gz -C /root/
 - cd /root/cookbooks/firewall
